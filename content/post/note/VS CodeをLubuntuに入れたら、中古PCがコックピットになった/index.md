Lubuntuという軽量な土台の上に、Visual Studio Code（VS Code）をインストールした。
その瞬間、単なる「古いパソコン」は、世界へ繋がる「コックピット」へと進化した。

左側にはブログの構造が並び、下側にはターミナルが鎮座する。
Markdownで文字を打てば、右側でリアルタイムに記事が組み上がっていく。

派手なPCは必要ない。
無料のOSと、最高の無料エディタ。そして自分の思考。
それさえあれば、この狭い部屋、中古の格安PCからでも、小さな氷を海に浮かべることができる。

今、私の目の前には、最高の執筆環境が整っている。
あとは、書き出すだけだ。
